Question-1: What are the different ways to select an element in the DOM?

getElementById: Selects an element by its unique ID (e.g., document.getElementById("myId")).
getElementsByClassName / getElementsByTagName: Selects elements by class or tag name, returning an HTMLCollection.
querySelector / querySelectorAll: Selects elements using CSS selectors, where querySelector returns the first match and querySelectorAll returns a NodeList of all matches.F

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Returns or sets the HTML content inside an element, including HTML tags (e.g., element.innerHTML = "Bold" renders bold text).
innerText: Returns or sets only the visible text inside an element, ignoring hidden elements and preserving formatting.
textContent: Returns or sets all text inside an element, including hidden elements, without formatting or parsing HTML.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where a single event listener is added to a parent element to handle events on its child elements. Instead of adding event listeners to multiple child elements individually, the parent listens for events using event bubbling and determines the target element using event.target. This improves performance and is useful for dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a phase in the DOM event propagation process where an event starts from the target element and moves upward through its ancestors (parent, grandparent, etc.) until it reaches the document object.

Question-5: How do you create, add, and remove elements using JavaScript?

Create an Element: Use document.createElement(tagName).
Add an Element: Use appendChild(), append(), or insertBefore().
Remove an Element: Use remove() or removeChild().
